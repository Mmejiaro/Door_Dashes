Controls: 
	- Jump —> Up Arrow Key
	- Move Right —> Right Arrow Key
	- Move Left —> Left Arrow Key
	- Attack —> Space Bar
	- Interact with Doors —> Enter Key
	- Pick Up Objects —> E

Cheat Codes:
	- Alt S —> Jump to the Shop Room
	- Alt B —> Jump to Boss Room
	- Alt D —> Destroy instances of enemies 
	- Alt L —> Receive 10 lives
	- Alt G —> Receive 150 Gold
